# Claude Code 源代码学习

> 一本深入剖析 Claude Code CLI 工具源代码的技术书籍

---

## 书籍简介

本书系统性地分析了 Claude Code CLI 工具的完整源代码，涵盖架构设计、核心机制、工具系统、服务层、终端UI等方方面面。全书共 50 章，约 10 万字，包含 50+ 个 Mermaid 架构图表。

**适合读者**：
- CLI 工具开发者
- TypeScript/React 工程师
- AI 工具架构研究者
- Claude Code 深度用户

### 入口与启动

- `src/Tool.ts` - Tool 接口定义

- `src/types/command.ts` - Command 类型

### 状态管理
- `src/state/AppStateStore.ts` - AppState 存储
- `src/state/AppState.tsx` - React Provider

### 工具实现
- `src/tools/BashTool/` - Shell 执行
- `src/tools/FileEditTool/` - 文件编辑
- `src/tools/AgentTool/` - 多 Agent
- `src/tools.ts` - 工具聚合

### 服务层
- `src/services/api/claude.ts` - API 客户端
- `src/services/mcp/` - MCP 服务
- `src/services/compact/` - 压缩算法
