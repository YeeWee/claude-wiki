# 搜索工具集

搜索工具是 Claude Code 进行代码库探索的核心能力，基于 ripgrep 实现 GlobTool 和 GrepTool。

## 核心内容

### GlobTool 文件匹配

基于 ripgrep `--files` 模式：
- 参数：pattern（glob 模式）、path（搜索目录）
- 结果按修改时间排序，默认限制 100 个文件
- 相对路径转换节省 token
- 环境变量控制：CLAUDE_CODE_GLOB_NO_IGNORE、CLAUDE_CODE_GLOB_HIDDEN

### GrepTool 内容搜索

三种输出模式：
- files_with_matches：返回匹配文件列表，按 mtime 排序
- content：返回匹配行内容，包含行号
- count：返回每个文件的匹配次数

核心参数：
- pattern、path、glob、type（文件类型）
- output_mode、-i（大小写）、-A/-B/-C（上下文）
- head_limit/offset 分页机制

### ripgrep 集成

三种运行模式：
- system：系统安装的 rg
- embedded：bun-internal 内嵌
- builtin：vendor 目录二进制

特殊处理：
- EAGAIN 错误自动重试单线程模式
- WSL 平台延长超时时间（60s）
- macOS 自动代码签名

### 性能优化

- 结果限制防止 token 爆炸
- 相对路径转换节省 token
- 配置和文件计数使用 memoize 缓存
- 文件计数四舍五入保护隐私
- 流式处理支持实时结果

## 设计启示

- 利用成熟底层工具（ripgrep）构建智能抽象层
- 多种输出模式满足不同搜索需求
- 自动排除版本控制目录
- 分页机制处理大型结果集
