# 文件操作工具集

文件操作是 Claude Code 最基础也是最核心的能力之一，三大工具构成完整闭环：Read、Edit、Write。

## 核心内容

### FileReadTool

多格式文件读取：
- 支持文本、图像、PDF、Jupyter Notebook
- 参数：file_path（必须绝对路径）、offset、limit、pages（PDF）
- 输出使用 cat -n 格式添加行号
- Token 限制与安全机制：文件大小限制、二进制文件检测、设备文件阻塞检测
- 去重机制：避免重复发送相同内容

### FileEditTool

精确字符串替换编辑：
- 参数：file_path、old_string、new_string、replace_all
- 验证流程：8 重检查（字符串相同、权限、文件大小、存在性、读取状态、修改时间、字符串匹配、多次匹配）
- 引号规范化：处理弯引号和直引号差异，保持文件引号风格
- 原子执行：加载-编辑-写入之间避免 async 操作

### FileWriteTool

完整文件覆盖或创建新文件：
- 明确区分 create 和 update 模式
- 验证流程：secrets 检查、权限检查、文件存在性、读取状态、修改时间
- 行尾处理：使用 LF（模型意图），Edit 保持原行尾风格

### readFileState 缓存

核心安全机制：
- FileState 结构：content、timestamp、offset、limit、isPartialView
- LRU 缓存：双重限制（条目数 + 大小）
- 路径规范化确保缓存一致性
- 生命周期：读取时设置、编辑/写入后更新

## 设计启示

- 先读后写保证操作安全性
- 原子操作防止并发冲突
- 引号规范化处理模型输出差异
- LRU 双重限制防止内存膨胀
