## 核心术语表（部分）

| 术语 | 英文名 | 定义 |
|------|--------|------|
| **Tool** | Tool | AI 可执行操作的核心接口 |
| **Command** | Command | 斜杠命令类型定义 |
| **Skill** | Skill | 通过 SKILL.md 文件定义的技能 |
| **Hook** | Hook | 生命周期钩子 |
| **MCP** | Model Context Protocol | 模型上下文协议 |
| **AppState** | AppState | 全局状态管理对象 |
| **PermissionMode** | PermissionMode | 权限模式枚举 |
| **Sandbox** | Sandbox | 沙箱安全机制 |
| **Feature Flag** | Feature Flag | 构建时功能开关 |
| **Fast Path** | Fast Path | CLI 启动优化机制 |
| **Fail-close** | Fail-close | 默认拒绝的安全策略 |
