MCP（Model Context Protocol）是 Anthropic 开发的开放协议，用于建立 AI 模型与外部工具、资源之间的标准化连接。Claude Code 通过 MCP 服务集成实现了与各种外部系统的无缝交互。

## 核心要点

MCP 在 Claude Code 中扮演"能力扩展器"的角色，允许：
- 调用外部工具执行特定任务
- 访问远程资源和数据源
- 通过 Elicitation 机制与用户进行交互式确认
- 动态加载插件提供的服务

## 协议基础

MCP 基于 JSON-RPC 2.0 协议，定义了四大核心概念：
1. **工具（Tools）**: 可被调用的函数，执行特定操作
2. **资源（Resources）**: 可被读取的数据源
3. **提示（Prompts）**: 预定义的提示模板
4. **能力（Capabilities）**: 服务器声明支持的功能

## 传输类型

Claude Code 支持六种传输方式：
- **Stdio**: 通过标准输入/输出流与本地进程通信
- **SSE**: Server-Sent Events，用于 HTTP 服务器持久连接
- **HTTP**: Streamable HTTP，符合 MCP 规范的现代传输
- **WebSocket**: 双向实时通信
- **进程内传输**: 避免启动重量级子进程（如 Chrome MCP）
- **SDK 控制传输**: 桥接 SDK 进程与 CLI 进程

## 连接生命周期

连接管理包含完整的生命周期：
- 初始化握手和超时处理
- 错误恢复和重连机制（指数退避）
- 优雅关闭和资源清理

## Elicitation 交互

支持两种模式的用户交互：
- **Form 模式**: 服务器请求结构化数据输入
- **URL 模式**: 引导用户访问 URL 并等待确认

## 认证支持

- OAuth 认证用于远程 MCP 服务器
- Claude.ai 代理认证包装
- needs-auth 状态缓存机制

## 配置管理

多层级配置优先级：
local > user > project > dynamic > enterprise > claudeai > managed

基于签名的去重机制避免重复服务器配置。
