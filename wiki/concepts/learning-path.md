学习路径是 Claude Code 从入门到专家的五阶段成长路线。

## 五阶段

| 阶段 | 时间 | 核心目标 |
|------|------|---------|
| 入门 | 1-2 周 | 安装配置、基本命令、权限理解 |
| 基础 | 2-4 周 | 工具系统、技能创建、MCP 基础 |
| 进阶 | 4-8 周 | Hook 系统、复杂技能、MCP 开发 |
| 专家 | 持续 | 源代码分析、架构理解、定制扩展 |
| 贡献 | 持续 | 开源贡献、社区分享、最佳实践 |
